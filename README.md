# Quiz App

A responsive and interactive Quiz Application built using HTML, CSS, and JavaScript. This project allows users to test their knowledge through multiple-choice questions, receive instant feedback, and view their final score at the end of the quiz.

## Features

* Multiple-choice questions
* Real-time answer validation
* Score calculation and display
* Responsive design for mobile and desktop devices
* User-friendly interface
* Restart quiz functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Overview

The Quiz App provides an engaging way for users to assess their knowledge on various topics. Each question is presented with multiple answer options, and users can select their answers and move through the quiz. At the end, the application calculates and displays the final score.

## How to Run

1. Clone the repository:
   git clone https://github.com/your-username/quiz-app.git

2. Navigate to the project folder:
   cd quiz-app

3. Open `index.html` in your browser.
